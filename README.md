# Bell Study Guide (Vercel)

This project is configured for static deployment on Vercel.

## What is configured

- `vercel.json` rewrites `/` to `Bell_Full_Study_Guide.html`
- `index.html` exists for local static-server compatibility

## Deploy steps

1. Push this repository to GitHub.
2. In Vercel, click **Add New → Project**.
3. Import this repository.
4. Keep defaults (Framework Preset: **Other**).
5. Click **Deploy**.

After deploy, your root URL (`/`) will open the full study guide page.
